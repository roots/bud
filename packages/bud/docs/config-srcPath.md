# bud.srcPath

Sets the directory containing a project's source files and assets.

Once set, this directory may be utilitized using [bud.src](config-src.md).

## Usage

```js
bud.srcPath('resources/assets')
```

## Arguments

Name | Type |
------ | ------ |
`path` | string |
