# bud.distPath

Sets the directory where assets will be built to. By default this directory is set as `dist` (relative to the `projectPath`).

This directory is utilitized by [bud.dist](config-dist.md).

## Usage

```js
bud.distPath('build')
```

## Arguments

| Name   | Type   |
| ------ | ------ |
| `path` | string |
