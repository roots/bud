## Facade.minimize property

Enables minification of built assets.

Signature:

```typescript
minimize: minimize;
```

## Example 1

Enable:

```js
bud.minimize();
```

## Example 2

Explicitly disable:

```js
bud.minimize(false);
```

## Example 3

Explicitly enable:

```js
bud.minimize(true);
```
