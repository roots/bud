# @roots/bud-solid

[Solid](https://solidjs.org) support can be added by installing the **@roots/bud-solid** extension.

## Installation

<Tabs
  defaultValue="yarn"
  values={[
    {label: 'yarn', value: 'yarn'},
    {label: 'npm', value: 'npm'},
  ]}>
  <TabItem value="yarn">

    yarn add @roots/bud-solid --dev

  </TabItem>
  <TabItem value="npm">

    npm install @roots/bud-solid --save-dev

  </TabItem>
</Tabs>

## Installing peer dependencies

After installing the main package, run the `bud init` command to install required peer dependencies:

<Tabs
  defaultValue="yarn"
  values={[
    {label: 'yarn', value: 'yarn'},
    {label: 'npm', value: 'npm'},
  ]}>
  <TabItem value="yarn">

    yarn bud init

  </TabItem>
  <TabItem value="npm">

    npm run bud init

  </TabItem>
</Tabs>

## Usage

Include **@roots/bud-solid** in your config. Make sure it is included after [@roots/bud-babel](/extensions/bud-babel).

```js
bud.use([
    require('@roots/bud-babel'),
    require('@roots/bud-solid'),
])
```
