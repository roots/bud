# @roots/bud-esbuild

[Esbuild](https://esbuild.github.io) support can be added by installing the **@roots/bud-esbuild** extension.

:::warning

Esbuild does not include support for hot module reloading.

:::

## Installation

<Tabs
  defaultValue="yarn"
  values={[
    {label: 'yarn', value: 'yarn'},
    {label: 'npm', value: 'npm'},
  ]}>
  <TabItem value="yarn">

    yarn add @roots/bud-esbuild --dev

  </TabItem>
  <TabItem value="npm">

    npm install @roots/bud-esbuild --save-dev

  </TabItem>
</Tabs>

## Usage

```js
bud.use('@roots/bud-esbuild')
```

## Configuration

Set options:

```js
bud.esbuild.setOptions({target: 'es2020'})
```
