# @roots/bud-preset-wordpress

The [@roots/bud-preset-wordpress](/extensions/presets/bud-preset-wordpress) package is a great starting point for WordPress plugins & themes.

If you plan on using it in a WordPress theme you should consider using [@roots/sage](https://github.com/roots/sage), which is a best-practice grade
solution to WordPress theming. It leverages this extension internally.

`@roots/bud-preset-wordpress` includes out-of-the-box support for:

- Babel
- PostCss
- React
- WordPress asset manifests and dependencies management

## Installation

  defaultValue="yarn"
  values={[

  <TabItem value="npm">

  </TabItem>

## Installing peer dependencies

After installing the main package, run the `bud init` command to install required peer dependencies:

<Tabs
  defaultValue="yarn"
  values={[
    {label: 'yarn', value: 'yarn'},
    {label: 'npm', value: 'npm'},
  ]}>
  <TabItem value="yarn">

    yarn bud init

  </TabItem>
  <TabItem value="npm">

    npm run bud init

  </TabItem>
</Tabs>

## Usage

Include **@roots/bud-preset-wordpress** in your config.

```js {4}
bud.use([require('@roots/bud-preset-wordpress')])
```
