# The bud CLI

:::note

CLI documentation is a work-in-progress

:::

For full documentation you can use the `--help` flag:

```shell
bud --help
```

## bud build

Run a build in `production` mode.

```shell
bud build
```

## bud dev

Run a build in `development` mode.

```shell
bud dev
```

## bud init

Install required peer dependencies for any extensions.

```shell
bud init
```
