## Install bud

Start by installing **@roots/bud** and **@roots/bud-cli**.

<Tabs
  defaultValue="yarn"
  values={[
    {label: 'yarn', value: 'yarn'},
    {label: 'npm', value: 'npm'},
  ]}>
  <TabItem value="yarn">

    yarn add @roots/bud @roots/bud-cli

  </TabItem>
  <TabItem value="npm">

    npm install @roots/bud @roots/bud-cli --save-dev

  </TabItem>
</Tabs>

**@roots/bud** is the build tool itself. **@roots/bud-cli** provides us a CLI interface for running builds and managing our project.

## Building the app

:::success

Nope, you didn't miss a step. No configuration is needed as long as there is an entrypoint defined at **src/index.js**.

:::

<Tabs
  defaultValue="yarn"
  values={[
    {label: 'yarn', value: 'yarn'},
    {label: 'npm', value: 'npm'},
  ]}>
  <TabItem value="yarn">

    yarn bud build

  </TabItem>
  <TabItem value="npm">

    npm run bud build

  </TabItem>
</Tabs>

You should see your built file in **dist/main.js** (along with a file called **manifest.json**).

```shell {2-4}
  ├── package.json
  ├── dist
  │   ├── main.js
  │   └── manifest.json
  └── src
      ├── index.js
      └── index.css
```

## Adding css

Looks like our css is missing! Let's add it by importing it into the 'src/index.js' entrypoint. 

We'll review another way to handle this in the next part of the guide. But this will do us nicely for now.

```js {1} title='src/index.js'
  import './index.css'

  console.log('hello world!')
```

Rerun the build with **bud build**. You should see the compiled css appear in **dist**:

```shell {4}
  ├── package.json
  ├── dist
  │   ├── main.js
  │   ├── main.css
  │   └── manifest.json
  └── src
      ├── index.js
      └── index.css
```
