Our project is coming together, but it's a little rough working with vanilla js. Let's add support for [Babel](https://babeljs.io/docs/en/#babel-is-a-javascript-compiler).

We can do this easily using the [**@roots/bud-babel extension**](/extensions/bud-babel).

## Installing an extension

Start by installing [**@roots/bud-babel**](/extensions/bud-babel).

<Tabs
  defaultValue="yarn"
  values={[
    {label: 'yarn', value: 'yarn'},
    {label: 'npm', value: 'npm'},
  ]}>
  <TabItem value="yarn">

    yarn add @roots/bud-babel --dev

  </TabItem>
  <TabItem value="npm">

    npm install @roots/bud-babel --save-dev

  </TabItem>
</Tabs>

## Installing extension dependencies

Extensions might not come with batteries included.

It's a good idea after installing a new extension to run [bud init](/guides/bud-cli/extensions#bud-extensionsinstall) to install any missing peer dependencies:

<Tabs
  groupId="pacman"
  defaultValue="yarn"
  values={[
    {label: 'yarn', value: 'yarn'},
    {label: 'npm', value: 'npm'},
  ]}>
  <TabItem value="yarn">

    yarn bud install

  </TabItem>
  <TabItem value="npm">

    npm run bud install

  </TabItem>
</Tabs>

This guarantees our project meets the requirements of our installed extensions.

:::info Protip

It's also a good idea to run [bud install](/guides/bud-cli/extensions#bud-extensionsinstall) after upgrading extensions in case requirements have changed.

:::

Your scripts will now be transpiled with babel.

You can see for yourself by modifying `src/index.js` so that it contains some fancy stuff:

```js title='src/index.js'
class CoolClass {
  message = 'hello world'

  function sayHello() {
    return this.message
  }
}

const cool = new CoolClass()

console.log(cool.sayHello())
```

Run a build and check `dist/main.js`. You should find the class declaration has been transpiled away.
