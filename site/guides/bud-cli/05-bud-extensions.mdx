## bud extensions:install

Installs any packages registered as dependencies by extensions listed in `package.json`.

<CodeBlock title="$ yarn bud extensions:install --help">
  {InitOutput}
</CodeBlock>

Aliases: `bud init`

## bud extensions:list

List all extensions installed to a project

```shell title="$ yarn bud extensions:list --help"
List extensions available to project

USAGE
  $ bud extensions:list

EXAMPLE
  $ bud extensions:list
```
