bud.isDevelopment

Boolean property which is true when bud.mode is development

Usageâ€‹

Very useful in conditionals:

bud.config.js
module.exports = bud =>
  bud.isDevelopment && bud.devtool('source-map')

Pairs well with bud.when:

bud.config.js
module.exports = bud =>
  bud.when(bud.isDevelopment, bud => bud.devtool())
