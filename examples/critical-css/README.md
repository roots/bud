<h1 align="center">
  <strong>Example: Critical CSS</strong>
</h1>

## Requirements

- `@roots/bud`
- `@roots/bud-cli`
- `@roots/bud-criticalcss`

This project also uses `tailwindcss` so that there is something to make the critical css from. So, to use this as-is you'll need to additionally install:

- `@roots/bud-postcss`
- `@roots/bud-tailwindcss`

## Installation

```sh
# Install core dependencies
yarn add @roots/bud @roots/bud-cli @roots/bud-criticalcss --dev

# Optionally
yarn add @roots/bud-postcss @roots/bud-tailwindcss --dev

# Install other dependencies
yarn bud extensions:install
```
