# @roots/bud-tailwindcss

[TailwindCSS](https://tailwindcss.org) support can be added by installing the **@roots/bud-tailwindcss** extension.

## Installation

<Tabs
  defaultValue="yarn"
  values={[
    {label: 'yarn', value: 'yarn'},
    {label: 'npm', value: 'npm'},
  ]}>
  <TabItem value="yarn">

    yarn add @roots/bud-tailwindcss --dev

  </TabItem>
  <TabItem value="npm">

    npm install @roots/bud-tailwindcss --save-dev

  </TabItem>
</Tabs>

## Installing peer dependencies

After installing the main package, run the `bud init` command to install required peer dependencies:

<Tabs
  defaultValue="yarn"
  values={[
    {label: 'yarn', value: 'yarn'},
    {label: 'npm', value: 'npm'},
  ]}>
  <TabItem value="yarn">

    yarn bud init

  </TabItem>
  <TabItem value="npm">

    npm run bud init

  </TabItem>
</Tabs>
