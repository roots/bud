## Installation

Install **@roots/wordpress-theme-json-webpack-plugin** to your project.

Yarn:

```sh
yarn add @roots/wordpress-theme-json-webpack-plugin --dev
```

npm:

```sh
npm install @roots/wordpress-theme-json-webpack-plugin --save-dev
```
