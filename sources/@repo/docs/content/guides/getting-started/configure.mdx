:::tip

Hey, did you know? [No configuration is needed](/guides/config/zero-config) as long you have something at **src/index.js**.

:::

Create a file titled **bud.config.js** file in the project root.

This file exports a single function that will tell **bud.js** what to do in order to build our project.

}
