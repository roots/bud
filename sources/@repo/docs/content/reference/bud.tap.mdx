Access the bud object through a callback. Useful to maintain a function chain.

## Usage

```js title=bud.config.js
export default async bud => {
  bud.tap(bud => {
    bud.entry('index.js')
  })
}
```

It may be helpful to consider that the `bud.config.js` export is esssentially just a **bud.tap** callback.
