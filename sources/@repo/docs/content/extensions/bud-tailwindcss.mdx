# @roots/bud-tailwindcss

[TailwindCSS](https://tailwindcss.org) support can be added by installing the [@roots/bud-tailwindcss](/extensions/bud-tailwindcss) extension.

## Installation

[@roots/bud-tailwindcss](/extensions/bud-tailwindcss) requires [@roots/bud-postcss](/extensions/bud-postcss) in order to function.

<Install packages="@roots/bud-tailwindcss @roots/bud-postcss" />
