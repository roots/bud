:::info

`bud.wpjson` methods don't return `bud`. You can't just continue chaining your config calls after using them.

You should either use [bud.tap](https://bud.js.org/docs/bud.tap):

```ts
bud
  .setPublicPath('/app/themes/sage/public/')
  .tap(bud => bud.wpjson.useTailwindColors().enable())
  .serve(3000) // chain continues
```

or, break the chain:

```ts
bud.setPublicPath('/app/themes/sage/public/')
bud.wpjson.useTailwindColors().enable()
bud.serve(3000) // c
```

:::
