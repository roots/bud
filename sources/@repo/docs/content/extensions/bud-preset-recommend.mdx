The [@roots/bud-preset-recommend](/extensions/bud-preset-recommend) package is a great starting point for a wide variety of projects.

It includes the following extensions:

- [@roots/bud-babel](/extensions/bud-babel)
- [@roots/bud-postcss](/extensions/bud-postcss)

## Installation

```sh npm2yarn
npm install @roots/bud-preset-recommend --save-dev
```
