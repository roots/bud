# @roots/bud-react

[React](https://reactjs.org) support can be added by installing the [@roots/bud-react](/extensions/bud-react) extension.

## Installation

[@roots/bud-react](/extensions/bud-react) requires [@roots/bud-babel](/extensions/bud-babel) in order to function.
