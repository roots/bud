SSL support and incremental improvements and fixes for core bud features.

<!--truncate-->

## SSL support for bud server

Bud server now supports `https` connections!

```ts title="bud.config.js"
bud.serve({
  url: 'https://example.test',
  cert: '/path/to/cert.crt',
  key: '/path/to/key.key',
})
```

For more on how to set this up [check out the bud.serve documentation](/docs/bud.serve).

## `theme.json` support for @roots/sage

You can generate a `theme.json` file for your theme using the `@roots/sage` package.

```ts title="bud.config.js"
app.themeJson({
  typography: {
    customFontSize: false,
  },
})
```

You can even sync your tailwind palette:

```ts title="bud.config.js"
app.useTailwindColors()
```

Find out more in the updated docs for [@roots/sage](/extensions/sage).
