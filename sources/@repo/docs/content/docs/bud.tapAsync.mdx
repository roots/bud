Access the bud object through an async callback. Useful to maintain a function chain.

## Usage

```js title=bud.config.mjs
export default async bud => {
  await bud.tapAsync(bud => {
    /* Do something with the bud object */
  })
}
```

It may be helpful to consider that the bud.config.mjs export is esssentially just a `bud.tapAsync` callback.
