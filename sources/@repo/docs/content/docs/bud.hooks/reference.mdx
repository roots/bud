# Hooks reference

:::note

Docs are a work-in-progress.

:::

This page is very difficult to keep in sync with source. To furhter explore what hooks are available it is recommended
to just explore the `bud.hooks` object directly.

```ts
console.log(bud.hooks.all())
```
