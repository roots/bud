Set credentials.

Credentials should be supplied as an object with the following properties:

- `accessKeyId` - AWS access key ID
- `secretAccessKey` - AWS secret access key

## Usage

```typescript
bud.fs.setCredentials({accessKeyId: `***`, secretAccessKey: `***`})
```
