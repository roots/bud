Specify if a build is being run in a `production` or `development` context.

[bud.mode](/docs/bud.mode/) will always be either `production` or `development`.

When running bud [using the cli](/guides/cli/):

- [bud build](/guides/cli/build) will always run in `production`.
- [bud dev](/guides/cli/build) will always run in `development`.

## Additional information

[bud.mode](/docs/bud.mode/) correlates with `NODE_ENV` and `BABEL_ENV`.
