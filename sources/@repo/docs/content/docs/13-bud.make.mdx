Create a new, configurable instance of Bud.

For more context on how this might be useful check out [the guide on multi-compiler configurations](/guides/general-use/multi-compiler).

## Usage

**bud.make** is an asyncronous function that takes two parameters:

- The **name** of the new compiler
- An optional callback to use for configuring the compiler.

```js
await bud.make('scripts', async child =>
  child.entry('app', 'app.js'),
)
```

It returns a `Promise` which resolves to the root instance for further chaining.
