Interface Hooks

Assign and filter callback to values.

example

Add a new entry to the webpack.externals configuration:

hooks.on(
  'build/externals',
  externals => ({
    ...externals,
    $: 'jquery',
  }),
)
example

Change the webpack.output.filename format:

hooks.on(
  'build.output.filename',
  () => '[name].[hash:4]',
)
